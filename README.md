# Electric Vehicle Analytics Dashboard

## Overview

The Electric Vehicle Analytics Dashboard is a data visualization project developed using Python Flask, HTML, and Tableau.

The application analyzes Electric Vehicle data such as battery capacity, driving range, charging time, efficiency score, safety rating, price, and top speed.

Users can explore the dashboard through a simple web interface.
